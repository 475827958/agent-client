# 项目期望
我想开发一套桌面版的agent文档软件，类似workbuddy桌面应用的界面，并且支持打包安装win11电脑

# 技术要求
React
+ Electron
+ Vite
+ TypeScript
+ Monaco Editor


# 界面要求
第一版：类似workbuddy桌面，左侧展示 任务（聊天记录），设置（模型api的key设置） ，右侧即聊天输出内容 和 输入框（支持 @ 引用对话文件 ，/ 调用技能与指令）；

# 功能要求
1：类似workbuddy桌面应用的界面，问答前可以选中本地文件夹或者文件进行问答，支持允许完全访问权限开关
2：可以支持模型api key配置，对接大模型。并且类似claude（需要后期接口sse推送），有自己计划步骤，并且输出需要权限时候提示 操作  是 否 等和目前市场上通用agent一样的提示功能（需要后期SSE 接口推送消息）。
3：支持选择工作空间，用于生成代码或者读取代码的地方
4：本产品是纯前端。agent 、模型等是后端服务控制，用于企业级应用管理。
5：其中包括根据输入内容，大模型回答后，执行选择的工作空间文件夹内容进行CURD操作，
agent的本地对文件操作的工具，glob read grep write edit 这五个工具
6：可打包可本地运行的应用
